Hypothesis!!

`H_a = tt"Alternative Hypothesis"`
`H_o = tt"NULL Hypothesis"`
`mu_i = tt"Mean for incongruent scores"`
`mu_c = tt"Mean for congruent scores"`
`x_D = tt"Desired difference between both the mean"`
`t_(stat) = tt"t-statistics for the sample"`

NULL Hypothesis

`H_o: mu_i - mu_c = 0 -> tt"There is no"` `tt"difference between"` `tt"both the means"`

ALTERNATIVE Hypothesis

`H_a: mu_i - mu_c ne 0 ->` `tt"There is significant"` `tt"difference between"` `tt"both the means for"` `tt"2 tailed t-test"`

`H_a: mu_i - mu_c lt 0 ->` `tt"Incongruent score"` `tt"mean is lesser"` `tt"than the congruent"` `tt"mean in 1"` `tt"tailed t-test"`
`H_a: mu_i - mu_c gt 0 ->` `tt"Incongruent score"` `tt"mean is greater"` `tt"than the congruent"` `tt"mean in 1"` `tt"tailed t-test"`

`t` Statictics at `alpha` value of 0.05

`t_(alpha = 0.05) = tt"12.4"`

Standard Error

`SE = sqrt(mu_i^2/n_1 + mu_c^2/n_2)`

t-statistics for the sample

`t_(stat) = ((mu_i - mu_c) - x_D) / (SE) = tt"1.2"` `iff x_D = 0`



Total degress of freedom

`df = (n_1 - 1) + (n_2 - 1) = tt"46"`
