Hypothesis!!

`H_a = tt"Alternative Hypothesis"`
`H_o = tt"NULL Hypothesis"`
`bar x_i = tt"Mean for incongruent scores"`
`bar x_c = tt"Mean for congruent scores"`
`bar x_d = tt"Point Estimate"`
`t_(stat) = tt"t-statistics for the sample"`
`delta = tt"The desired difference between both means"`

NULL Hypothesis

`H_o: mu_i - mu_c = 0 -> tt"There is no"` `tt"difference between"` `tt"both the means"`

ALTERNATIVE Hypothesis

`H_a: mu_c - mu_i lt 0 ->` `tt"Congruent score"` `tt"mean is lesser"` `tt"than the incongruent"` `tt"mean in 1"` `tt"tailed t-test"`

Total degress of freedom

`df = (n - 1) = tt"23"` `if n = tt"Sample Size"`

One tailed `t` value at `alpha` of 0.05

`t_(alpha = 0.05) = +-tt"1.714"`

Standard Deviation of the difference

`S = sqrt(sum_0^(n-1) (d_i - bar (md))^2 / (n -1)) = tt"1.8726"`
`iff bar (md) = sum_0^(n-1) (x_i - y_i) / n` `tt"= -7.9647"`

Point of Estimate

`bar x_d = bar x_c - bar x_i = bar (md) = tt"-7.9647"`

Standard Error

`SE = S / sqrt(n) = tt"0.3823"`

t-statistics for the sample

`t_(stat) = ((bar x_c - bar x_i) - delta) / (SE) =tt"-20.8365"` `iff delta = 0`

Cohen's d for effect size

`d = (bar x_c - bar x_i)/S = tt"4.2532"`

95% Cofidence Interval

`CI = bar x_d +-` `t_(alpha = 0.05)` `xx SE` ` => CI in tt"[-8.62, -7.31]"`

Coefficient of Determination `r^2`

`r^2 = t_(stat)^2 / (t_(stat)^2 + df) = tt"0.9497"` `~=tt"95%"`
